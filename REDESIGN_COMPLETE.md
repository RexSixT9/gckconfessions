## 💬 Messaging Updates

### Landing Page Changes (6 updates)

1. **Badge** - More specific about value
   - From: "Anonymous & Secure"
   - To: "Private • Safe • Moderated"

2. **Main Title** - More inclusive language
   - From: "Share Your Confessions"
   - To: "Share Your Inner Thoughts"

3. **Subtitle** - Clearer value proposition
   - From: "A safe, anonymous space..."
   - To: "A safe, judgment-free space..."

4. **Trust Indicators** - More direct benefits
   - From: "Spam-protected", "Community-driven"
   - To: "No spam • Protected", "Trusted by community"

5. **How It Works** - Proper capitalization
   - From: "How it works"
   - To: "How It Works"

6. **Steps Description** - Better clarity
   - Step 1: "authentic thoughts" instead of "confession"
   - Step 2: "Our team reviews" instead of "Our admins carefully review"
   - Step 3: "Share" instead of "Post"

7. **Final CTA** - More empowering
   - From: "Ready to share?"
   - To: "Ready to Share Your Story?"
   - Description emphasizes voice mattering

### Submit Page Changes (9 updates)

1. **Badge** - Privacy-focused messaging
   - "100% Private • Zero Tracing"

2. **Form Labels** - Proper capitalization
   - "Your Confession" (was lowercase)
   - "Companion Song" (was lowercase)

3. **Placeholders** - Better guidance
   - "What's on your mind? Be honest and authentic..."

4. **Helper Text** - Positive framing
   - "Be authentic and respectful. Hateful or spam content will be removed."

5. **Character Counter** - Clearer units
   - Now shows "X / Y characters" instead of just "X / Y"

6. **Success Message** - Community-focused
   - "Our community team will review your confession shortly"

7. **Error Message** - Clearer title
   - "Unable to Submit" instead of "Submission failed"

8. **Notice Styling** - Improved colors
   - Uses semantic color variables
   - Better contrast in both modes

9. **Song Field Hint** - Better context
   - "(optional — add a song that represents your confession)"

### Admin Page Changes (2 updates)

1. **Title** - More descriptive
   - "Manage Community Confessions"

2. **Subtitle** - Better verbs
   - "Review, approve, and curate authentic submissions"

## ✨ Visual Changes

### Badges & Accents
```
Before:
border-[hsl(var(--accent))]/30
bg-[hsl(var(--accent))]/5

After:
border-[hsl(var(--accent))]/40  ← Stronger border
bg-[hsl(var(--accent))]/8       ← More visible background
style={{boxShadow: '0 0 8px hsl(var(--accent))'}}  ← Added glow
```

### Notice Components
```
Before:
border-green-200/50
bg-green-50
text-green-700
dark:border-green-900/30
dark:bg-green-950/20

After:
border-[hsl(var(--success))]/40  ← Uses semantic variable
bg-[hsl(var(--success))]/8
text-[hsl(var(--success))]
dark:border-[hsl(var(--success))]/50  ← Better dark contrast
dark:bg-[hsl(var(--success))]/10
```

### CTA Sections
```
Before:
border-[hsl(var(--accent))]/20
bg-linear-to-br from-[hsl(var(--accent))]/5

After:
border-[hsl(var(--accent))]/30  ← 50% stronger
```

2. Follow existing contrast ratios
3. Test in both light/dark modes
4. Update documentation if needed

2. Avoid judgment/negative language
